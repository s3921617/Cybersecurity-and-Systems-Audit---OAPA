# Cybersecurity-and-Systems-Audit-OAPA README

This GitHub repository will serve as a document control/version control catalogue for the P000680CSITCP Team, for the project: RMIT Network and Systems Audit - OAPA.


##Key Deliverables
###Mature Network and Systems Audit
The primary objective of this project is to create a cybersecurity audit report for the client, Open Access Publishing Association (OAPA). This will be done by doing a structured network and systems audit for the client by searching the system for vulnerabilites. After this, a standard auditing procedure will take place, with risk assessments, improvment suggestions, and a finalized report.

###System Documentation Creation
The secondary objective of this project is to compile as much documentation as possible for the OAPA systems and backend on which the website operates, and create a comprehensive guide. The outcome will depend on how much access is provided to the system to the audit team. The final product will be a detailed documentation guide which will be beneficial to any technical personnel who will work with OAPA in the future.


##Purpose of GitHub
For the duration of this project, this GitHub repository will be used as a version control catalogue. 
As any major changes are made to either the audit report or OAPA system documentation guide, the changes will be pushed to this GitHub repository.
